# zhaoyajie.github.io

This repository contains the source code for my personal academic homepage.

## Files

- `index.html`: the homepage.
- `README.md`: this repository description.

## Deployment

Put `index.html` and `README.md` in the root of the GitHub Pages repository, then commit and push.

```bash
git add index.html README.md
git rm index.md   # optional: remove the old Markdown homepage if it still exists
git commit -m "Update personal homepage"
git push
```

After GitHub Pages finishes building, the site will be available at:

```text
https://zhaoyajie.github.io/
```
