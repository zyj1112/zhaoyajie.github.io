Problem (Russian Mathematical Olympiad 2000)
In a \(200\times 200\) board, every cell is colored black or white, and the difference between the numbers of black and white cells is \(404\). Prove that there exists a \(2\times2\) square containing an odd number of white cells.

Proof

Encode a white cell as \(1\) and a black cell as \(0\). Let \(a_{ij}\) be the color of the cell in row \(i\), column \(j\). Suppose, for contradiction, that every \(2\times2\) square contains an even number of white cells. Then

\[ a_{ij}+a_{i,j+1}+a_{i+1,j}+a_{i+1,j+1}\equiv 0\pmod 2. \]

Hence any two adjacent rows have the same changes between adjacent columns. Therefore each row is either identical to the first row or the complement of the first row.

Let the first row contain \(q\) white cells, and let \(p\) rows be identical to the first row. Then the total number of white cells is

\[ W=pq+(200-p)(200-q). \]

The number of black cells is \(40000-W\), so their difference is

\[ |40000-2W|=4|(100-p)(100-q)|. \]

By the condition, this equals \(404=4\cdot101\), so

\[ |(100-p)(100-q)|=101. \]

But \(p,q\in\{0,1,\dots,200\}\), hence \(|100-p|\le100\) and \(|100-q|\le100\), which makes this impossible. This contradiction proves that some \(2\times2\) square contains an odd number of white cells.
